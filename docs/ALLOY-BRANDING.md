# Alloy Workspace - Branding Guide

## Product Name
**Alloy Workspace** - A unified workspace suite for modern productivity.

## Tagline
"Create, modify, automate, and save anythingâ€”then share it anywhere."

## Core Value Proposition
Alloy Workspace unifies all standard workspace tasks in one app, giving users the best possible combination of power and simplicity instead of 6 or more separate apps with inconsistent features and restrictive sharing options.

## Key Brand Pillars

### 1. Universal
- Works on any platform, device, or OS
- No lock-in or format restrictions
- Data portability across ecosystems

### 2. Unified
- One app replaces multiple tools
- Consistent interface and workflows
- Seamless data flow between editors

### 3. Powerful
- Node-based visual editor (Blender-like)
- Automation and workflow orchestration
- Cross-ecosystem integration

### 4. Simple
- Intuitive, operational node categories
- Familiar UI patterns (Google/Microsoft style)
- Straightforward mental model

### Color Palette

### Node Categories (Operational Language)
- **Collect**: Bring information in
- **Edit**: Transform and enrich
- **Route**: Decide and control flow
- **Save**: Persist and store
- **Send**: Deliver outputs
- **Automate**: Schedule and trigger

## Messaging
- **For Enterprise**: "Replaces Google Workspace, Microsoft 365, and specialized tools."
- **For Creators**: "One canvas for everything. All your tools. One powerful app."
- **For Power Users**: "Universal platform with no platform restrictions."

## Target Audience
- Google Workspace / Microsoft 365 users seeking consolidation
- Power users frustrated with multi-app workflows
- Organizations seeking format-agnostic data management
- Teams requiring cross-ecosystem automation
